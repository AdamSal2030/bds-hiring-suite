2 Data entry

Build a spreadsheet from these loads.

Create an Excel file on this computer, enter the loads below into it, save it with your name, and upload it here. This shows us how you set up and organize data in a real spreadsheet.

How to do this
  1. Open a new spreadsheet. On this computer, open Microsoft Excel (or Google Sheets) and start a blank workbook.
  2. Type the column headings in row 1. In the very first row, type these exact headings, one per column, left to right:
    Load #|Origin|Destination|Equipment|Weight (lbs)|Miles|Rate/mi
  3. Enter the loads. Read each dispatch note below and type its details into a new row under the headings, one load per row. Put each value in the matching column.
  4. Save the file with your name. Save it as an Excel file (.xlsx). In Excel: File → Save As, choose Excel Workbook (*.xlsx), and name it:
    Your Full Name - Data Entry.xlsx
    Example: Sam Harris - Data Entry.xlsx. Tip: save it to the Desktop so it’s easy to find in the next step.

3 Email writing

Win the client over.

Read the scenario, then write an email to the customer (Sam) that wins him over and gets him to book with us. You can use the Claude chatbot to help. We want to see how clearly you write and how well you persuade.

The scenario

Sam asked us for a price to move a bulldozer from Dallas, TX to Kansas City, MO (about 550 miles). After checking the route, the fair all-in rate is $4,850, which covers the truck, fuel, the oversize permits, and a pilot car. Sam has hinted he wants the lowest number possible.

Your job: write an email to Sam that gives him the $4,850 price clearly and confidently, breaks down what is included so the number makes sense, convinces him it is worth it, and asks him to book with Badass Logistics.

Write the email

You can use the Claude chatbot on this computer to help. Give Claude the details from the scenario above and ask it to write a convincing email to Sam, then read it over and put it in your own words. Prefer to write it yourself? That works too. Either way, make sure the email:

  • gives Sam the $4,850 price clearly and with confidence,
  • explains what it covers so the number makes sense,
  • asks Sam to book with Badass Logistics.
Then send it
  1. Open Gmail in Chrome. It is already signed in as badasslogistics@gmail.com. Click Compose and send to info@badasslogistics.com.
  2. Write the email to Sam. Add a subject line and sign off with Regards, then your name and your email.
  3. Send it, then come back here and paste a copy below, then click I’ve sent the email.
